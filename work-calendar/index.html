Pensado para turnos que cambian

Un calendario limpio para consultar meses, mover jornadas y tener siempre presentes los días libres.

  • Vista mensual Consulta el mes completo con un diseño pensado para leer el calendario de un vistazo.
  • Vacaciones y permisos Marca vacaciones, permisos y descansos para tener el calendario laboral completo.
  • Horas extra Lleva el control de las horas añadidas y de las anotaciones más importantes.
  • Recordatorios Recibe avisos para cambios de jornada, fechas clave y ajustes de última hora.
  • Orden y contexto Todo lo esencial queda agrupado para que no tengas que reconstruir tu mes cada semana.
